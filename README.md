# Medical Professional Outreach System
### Sotheby's International Realty — Oakville, ON
**Property:** 3291 Harasym Tr, Oakville, Ontario

---

## Overview

A complete lead-generation and outreach management system targeting medical professionals across the GTA as potential buyers for a luxury Oakville listing. Built for CASL compliance using only publicly available business contact information from official registries.

---

## Quick Start

```bash
# 1. Install Python dependencies
pip3 install requests beautifulsoup4 pandas geopy

# 2. Scrape doctor registry (CPSO public)
python3 scrape_doctors.py

# 3. Scrape hospital gatekeeper contacts
python3 scrape_gatekeepers.py

# 4. Open the dashboard
# Option A — direct file open (works for most features):
open index.html

# Option B — run local server (recommended for data file loading):
python3 -m http.server 8080
# then open: http://localhost:8080
```

## Workflow

1. **Run scrapers** to populate the database (see Quick Start)
2. **Open dashboard** — the map, tables, and kanban load automatically
3. **Work through daily call list** on the Overview page (15 priority contacts each day)
4. **Track status** via the Kanban board — drag contacts through pipeline stages
5. **Use email templates** — copy and fill in contact-specific variables
6. **Add notes** on any doctor or gatekeeper record

## Doctor Scraper (`scrape_doctors.py`)

**Primary source:** CPSO (College of Physicians and Surgeons of Ontario) public registry  
https://www.cpso.on.ca/Public-Information-Services/Doctor-Search

**Target geography:**
- Halton: Oakville, Burlington, Milton, Georgetown
- Peel: Mississauga, Brampton
- Toronto: all M postal codes
- Hamilton: L8, L9 postal codes

**Data collected per doctor:**
- Full name, specialty
- Practice address, practice phone (business only)
- Hospital affiliation
- Year of registration
- Distance from 3291 Harasym Tr (calculated via geocoding)

**Technical notes:**
- Respects robots.txt
- 2-second delay between requests
- Falls back to representative GTA dataset if live scraping is blocked
- Uses `geopy` Nominatim for geocoding (free, no API key)

## CASL Compliance

This system is designed for full compliance with Canada's Anti-Spam Legislation (CASL):

**Basis for contact:** CASL s.6(5) — Implied Consent (Business-to-Business)

Implied consent applies when:
1. The recipient's business contact information is **publicly available** in a professional directory or registry
2. The sender clearly identifies themselves and their organization
3. The message is relevant to the recipient's business or professional role
4. Every message includes a clear and easy unsubscribe mechanism

**What this system does NOT collect:**
- Personal email addresses
- Residential phone numbers
- Home addresses

**What every outreach must include:**
- Your full legal name
- Your brokerage name and address
- A working phone number
- A statement offering to remove them from your list

**Required footer text for all emails:**
> *"To be removed from this list, simply reply to this email and I will action it immediately."*

## Technical Notes

- Dashboard is a single HTML file — no server required for embedded sample data
- All filter/sort/search state saved in localStorage
- Kanban board state persists in browser across sessions
- Map uses Leaflet.js + OpenStreetMap — no API key required
- If running from `file://` and data files don't load, the dashboard uses embedded sample data automatically
- For best results, serve via `python3 -m http.server 8080`
